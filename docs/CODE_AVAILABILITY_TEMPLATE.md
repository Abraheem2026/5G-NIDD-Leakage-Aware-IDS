# Code availability statement template

Use the following version after creating the public GitHub repository and archiving a release in Zenodo:

> The source code used for data auditing, preprocessing, feature selection, model training, holdout evaluation, fold-specific cross-validation, Seq/Offset ablation, paired statistical testing, controlled timing, and generation of the reported tables and figures is publicly available at [GitHub repository URL]. Version [release tag] is permanently archived in Zenodo under DOI [software DOI]. The repository provides the software dependencies, experimental configuration, random seeds, verified reference results, and instructions required to reproduce the analyses. The original 5G-NIDD dataset is not redistributed with the code and must be obtained separately from its official IEEE DataPort record (DOI: 10.21227/xtep-hv36).
